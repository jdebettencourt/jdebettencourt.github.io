CA County Severe Storms Map and Data Links

This page has links to a number of stormy weather choropleth maps for California Counties based on cumulative data from 2009 through 2018

The map demos use the LeafletJS web mapping library. The map shapes use geoJson structured data containing county boundaries or outlines. Data from the NWS storm data files have been aggregated into county level storm counts and are linked to county shapes as properties of each county.

For more information on these maps and associated data please contact James deBettencourt at james.s.debettencourt@gmail.com
